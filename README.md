Bu proje, temel bankacılık işlemlerini komut satırı üzerinden simüle eden dosya tabanlı bir hesap yönetim sistemidir. Amacı C dilinde struct, file i/o, kullanıcı doğrulama ve işlem mantığı gibi kavramları gerçekçi bir senaryo üzerinden uygulamaktır.
Sistem gerçek bir veritabanı veya ağ bağlantısı içermemektedir, tek kullanıcı çalışacak şekilde tasarlanmış eğitim amaçlı bir backend simülasyonudur.

Aktif Özellikler:
Şu anda projede sadece geçici hesap oluşturma işlemi aktiftir, program sonlandırıldığında veriler saklanmaz.
